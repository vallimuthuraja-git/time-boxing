# The Time Box - Daily Planner

A beautiful, digital daily planner web application inspired by traditional paper planners. Plan your day with priorities, time blocks, and notes in an elegant, printable interface.

## Features

- **12-Hour Time Format**: Clean AM/PM time slots from 5 AM to 11 PM
- **Multi-line Time Cells**: Each time slot can hold 2 lines of detailed notes
- **Daily Priorities**: Set up to 4 top priorities for the day
- **Brain Dump**: Large text area for capturing random thoughts and ideas
- **Auto-save**: Data persists in localStorage between sessions
- **Settings Menu**: Customize logo corner styles and access print functionality
- **Print-Ready**: Optimized styling for PDF generation and printing
- **Responsive Design**: Works on desktop and mobile devices

## Usage

1. Open `index.html` in a web browser
2. Fill in your daily date (auto-populated with today's date)
3. Add your top 4 priorities in the left column
4. Plan your time blocks for each half-hour slot
5. Use the Brain Dump area for additional notes

7. All changes auto-save locally

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- Local Storage API

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge) with Local Storage support.

## Customization

The planner's appearance and functionality can be customized by editing:

- `styles.css`: Visual styling, layout, colors
- `script.js`: Interactive behavior and data persistence
- `index.html`: Structure and content

## Print/PDF Export

Use the browser's print function (Ctrl+P) or the download option in the settings menu to generate a PDF version for printing.
